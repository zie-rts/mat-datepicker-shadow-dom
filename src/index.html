Datepicker ShadowDom Focus

The component is a custom element and uses ViewEncapsulation.ShadowDom.

Steps

  1. Open the datepicker popup via button (or focus the input element and press alt+down)
  2. Select a date => the popup will be closed
  3. The focus is lost

Here is the same custom element but with tabindex="-1" (focusable by JS). After the calender popup is closed, the webcomponent will be focused (red border).
